# ZenTask AI: Spec-Driven Distributed Todo System

## Mission
To bridge the gap between simple task management and cloud-native AI-orchestrated productivity.

## Core Pillars
1. **Intelligence**: Multi-modal (Text/Voice) and Multi-lingual (EN/UR) command processing.
2. **Connectivity**: Cloud-native distributed architecture with Kafka event streaming.
3. **Spec-Driven**: Every feature is defined by Markdown specs before implementation.

## Tech Stack
- **Frontend**: Next.js 16+, TypeScript, Tailwind CSS
- **Backend**: Python 3.13+, FastAPI, SQLModel
- **AI**: Gemini 2.5/3 Pro & Flash, SpeechRecognition API
- **Deployment**: Kubernetes (Helm), Dapr, Kafka
